# Connext Studio — Feedback & Issue Tracker

This repository is used to track **bug reports**, **feature requests**, and **general feedback** for Connext Studio

> **Note:** This repository does not contain source code. It exists solely for community issue tracking.

## How to Submit Feedback

1. Check [existing issues](../../issues) to see if your topic has already been reported.
2. If not, [open a new issue](../../issues/new/choose) using the appropriate template.

## Issue Categories

| Label | Use for |
|-------|---------|
| `bug report` | Something isn't working as expected |
| `feature request` | Feature requests or improvements |
| `other feedback` | General questions about Connext Studio |

## What to Include

When filing a bug report, please include:

- **Connext Studio version**
- **Operating system and version**
- **Steps to reproduce** the issue
- **Expected vs. actual behavior**
- **Screenshots or logs**, if applicable
